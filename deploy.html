Deploy to CloudFlare Deploy via GitHub

Manual Deployment Steps:

Log in to your CloudFlare dashboard
Navigate to Pages in the left sidebar
Click "Create a project" and connect your Git repository
Set build command to: npm run build
Set build output directory to: dist
Add environment variables as specified in the deployment guide
Click "Save and Deploy"

Note: For detailed instructions and advanced configuration options, please refer to the CLOUDFLARE_DEPLOYMENT_GUIDE.md file in your project.
